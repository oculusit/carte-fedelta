Perché FidAPPti?

📱

Scanner nativo

Scansiona barcode e QR code in tempo reale con la telecamera del tuo telefono, senza scattare foto.

💾

Funziona offline

Zero dipendenza dalla rete. I dati restano sul tuo dispositivo, sincronizzati solo se vuoi tu.

🔐

Nessuna registrazione

Apri e usa. Niente account, niente password, niente tracciamento. I tuoi dati sono tuoi.

⭐

Preferiti

Metti stelle alle carte che usi di più e ritrovale subito nella dashboard.

☁️

Backup cloud

Backup automatico opzionale su Supabase. Vuoi usare il tuo? Basta inserire URL e chiave anonima.

🎨

Loghi reali

Ogni carta mostra il logo del negozio. Proporne uno nuovo e gli admin lo approvano.

Privacy Policy

Ultimo aggiornamento: 2025

FidAPPti è concepita con la privacy come priorità fondamentale.

Dati raccolti

FidAPPti non raccoglie dati personali. Non esiste registrazione, non vengono inviati dati a server esterni senza il tuo consenso esplicito.

Dati memorizzati localmente

  • Numero della carta fedeltà e nome del negozio
  • Nome del titolare (opzionale)
  • Note personali

Tutti i dati restano esclusivamente sul tuo dispositivo.

Backup opzionale (Supabase)

Se scegli di attivare il backup cloud, i dati vengono criptati prima di essere inviati al tuo server Supabase personale.

Contatti

Per domande sulla privacy, contattaci all'indirizzo indicato nel repository GitHub del progetto.

Cookie Policy

Ultimo aggiornamento: 2025

FidAPPti non utilizza cookie di tracciamento, fingerprinting, analytics o pubblicità.

L'unica memorizzazione avviene in locale (localStorage / IndexedDB) per le impostazioni dell'app e il funzionamento offline.

Questo sito web non utilizza cookie di alcun tipo.
